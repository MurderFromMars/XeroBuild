<h3 align="center">XeroLinux Changelog</h3>

This month we discontinued the **Gnome Spin**. As of now, there will only be one to rule them all, the **KDE Plasma** ISO. Below are the changes.

#### Chages/Fixes

- Updated to KDE 6.5.2.
- Updated to latest Kernel.
- More Plasma Optimizations.
- More Panel / Dock Transparency.
- Added [**Kwin-Kzones**](https://github.com/gerritdevriese/kzones) Script.
- Enabled **Pacakge Chooser** in **Calamares**.
- Added [**Kurve Visualizer**](https://store.kde.org/p/2299506) widget.
- Reworked **Toolkit** from scratch using **Rust/GTK** UI.

**Notes :

> Make sure to connect to the Internet before launching the installer, will fail otherwise. Also the more packages you select from the chooser, the longer the install will take, it depends on your connection speed, so please be patient.
